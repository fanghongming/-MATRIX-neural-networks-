# -MATRIX-neural-networks-
矩阵和神经网络库（MATRIX  neural networks）
我们使用的是Eigen库，想要这个库，我们可以去https://eigen.tuxfamily.org/index.php?title=Main_Page
这个地方有文档，也可以在github上找到,https://github.com/PX4/eigen。
